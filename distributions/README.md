## Parameterisation reference distributions

This directory contains figures showing bonded interaction
distributions comparing CG simulations to their pseudo-CG (mapped atomistic)
counterparts. The figures have been generated using Fast-Forward's `ff_assess` program.

The files are labelled by the molecule's resname. Distributions for collective interactions 
shared by nucleotides are found in the [nucleotide_plots](nucleotide_plots) subdirectory.
